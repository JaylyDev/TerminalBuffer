# TerminalBuffer

A terminal text buffer that stores and manipulates displayed text built in Java.

## Features

- Stores and manipulates text in a grid of character cells in terminal
- Modifies character cell attributes such as foreground color, background color, and text styles (bold, italic, underline)
- Text wrapping and line breaking
- Cursor movement for inserting and deleting text

## Setup

> [!NOTE]
> This project has only been tested on Java 24.

### Clone the repository

```
git clone https://github.com/JaylyDev/TerminalBuffer.git
```

### Build

```
./gradlew build
```

### Run tests

```
./gradlew test
```
